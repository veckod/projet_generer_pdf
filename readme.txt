Ce projet en Php consiste à appliquer la class FPDF de la librairie PDFlib pour générer des fichiers PDF.
Dans ce stage On m'a chargé de générer deux types de documents dont une facture et une lettre.
Les données dans les documents doivent automatiquement correspondre à la DB. j'ai donc creer une db en mysql (voir le fichier "bdd_mysql.txt"). 
Il était demandé de reproduire les documents quasi à l'identique des deux exemples que vous pouvez 
trouver dans le dossier "/img
